# Portfolio Site

This directory contains a minimal static landing page for your AI Resume Analyzer project.

## How to use

1.  Customize `index.html` with your personal details and links.
2.  Host it for free on:
    - GitHub Pages
    - Vercel
    - Netlify
    - Or via your own Caddy/Nginx server.
